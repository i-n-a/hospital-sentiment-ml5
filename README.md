# Healthcare Sentiment Analysis with ml5.js

This project analyzes commentary feedback for a hospital in Lisbon using a custom sentiment analysis model built with ml5.js.

## Features

- Classifies comments as **positive**, **neutral**, or **negative**
- (Planned) Classifies comments into categories such as triage, waiting time, and staff
- Simple web interface for uploading new feedback datasets

## Tech Stack

- JavaScript / HTML / CSS
- ml5.js (neural network)
- GitHub Pages (optional, for hosting)

## Project Context

This project is a synergy between Master's in Marketing and Innovation & 
Master's Degree in Creative Computing and Artificial Intelligence.
Developed as part of a transdisciplinary project at IADE, combining **Machine Learning** and **Emerging Technologies in Marketing**.
